# FileArchiver + Huffman compression algorithm 

# This is my own file archiver (C)
## Input format:
*--file FILE* <br/>
*--create* <br/>
*--extract* <br/>
*FILE1 FILE2 ... FILEN* <br/>
## Examples:
***arc --file data.arc --create a.txt b.bin c.bmp*** <br/>
***arc --file data.arc --extract***<br/>
***arc --file data.arc --list***
